使用说明：

新安装就不提啦，简单的很= =

将本包内文件解压覆盖，然后进入数据库找到setting表，清空cloud这一行的内容(删除也行)，然后清空cache表
最后去系统设置刷新下看看云平台注册上没~如果有站点ID就成功了
